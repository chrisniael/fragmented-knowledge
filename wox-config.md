# Wox 配置

## 依赖

* .net framework >= 4.5.2
* [everything](https://www.voidtools.com), 64 bit 操作系统安装 64 bit 版本
* [python3](https://www.python.org/downloads/windows), 64 bit 操作系统安装 64 bit 版本

## 设置

#### 通用

* 开机自动启动
* 失去焦点时自动隐藏 Wox
* 隐藏任务栏图标 (隐藏后可以输入 settings 呼出)
* 语言：中文
* python path

    add it to %PATH% or set it in WoX settings
