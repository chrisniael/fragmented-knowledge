# Mac

## ldd in macOS

Mac 上显示二进制文件使用的动态库 (Linux ldd 的功能)

```bash
otool -L <binary-file>
```

## 快速显示词典释义

1. 选择文字后，触发键盘快捷键

   ```
   CMD + CTRL + D
   ```

2. 触摸板三指轻触选择的文字 (默认是 "单指用力点按")

   系统偏好设置 - 触摸板 - 光标与点按 - 查询与数据监测器  
   选择 "三个手指轻点"

3. 自定义快捷键

   系统偏好设置 - 键盘 - 快捷键 - 服务 - 在词典中查询  
   勾选并设置自定义快捷键
